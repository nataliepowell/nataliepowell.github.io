Rutgers Reimagined Concept + Design
A great university needs a great app.

Rutgers University–New Brunswick is home to over 50,000 students and the largest campus bus system in the United States. Its mobile application should be just as impressive. View on Figma

Goal: make it faster and easier to navigate campus.
announcement
people_alt
directions
Alert students when it is time to leave for class based on their current location, proximity to a bus stop, and traffic conditions.
Some students allow >45 minutes to get to class due to unpredictable bus complications.

Report student traffic and crowdedness.
With 50,000 students riding the busses, vehicle traffic isn’t the only traffic that slows down your commute.

Provide detailed directions that integrate the campus bus system.
Common navigation apps don’t include the Rutgers bus system in public transportation directions.
Implementation Ideas
The app hasn't been developed yet, but there are plenty of resources available to make it possible.

Rutgers APIs Rutgers maintains open-source APIs which include bus information, building locations, dining hall menus, and facility hours. MapQuest APIs MapQuest APIs will allow us to generate turn-by-turn directions and predict ETAs in real-time according to traffic.

Circuit Output Generator Course Project
Computer Architecture (CS211) project in C that generates a truth table for a given boolean logic circuit. Evaluates any combination of logical AND, OR, NAND, NOR, XOR, NOT, MULTIPLEXER, and DECODER gates. Input file can have a variable number of input, output, and temporary variables.
